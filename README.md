# ポモドーロタイマー (React + CDN)

ローカルでサーバを立てずに動かせる、React製のシンプルなポモドーロタイマーです。

## 使い方
1. `pomodoro/index.html` をブラウザで開きます（ローカルファイルとして開くだけで動作します）。
2. スタートを押すと「25分作業 → 5分休憩」を自動で繰り返します。
3. ストップで一時停止、リセットで25:00からの作業に戻ります。
4. モード切り替わり時のサウンドを鳴らす/鳴らさないをトグルで選べます。

## 技術メモ
- React / ReactDOM / Babel は CDN (unpkg) から読み込むため、npm インストール不要です。
- タイマー進行は `setInterval` で1秒ごとにカウントダウンし、残り1秒で作業/休憩モードを反転させています。
- CSS は `clamp` や `auto-fit` グリッドでスマホでも崩れにくいレスポンシブ構成です。
